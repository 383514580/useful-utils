# isBase64
判断是否base64编码, 多用于加载图片的时候做兼容判断.

## 函数参数 
| 参数名 | 说明 | 默认值 | 数据类型 |
| --- | --- | --- | --- |
| string | 任意字符串 | - |string|

## 函数返回
| 说明 | 数据类型 |
| --- | --- |
| 是否base64编码 | boolean |

## 实例 
``` javascript
isBase64('https://avatars1.githubusercontent.com/u/8264787?s=460&v=4'); // false
```
