# angle2rad
角度转弧度

## 函数参数 
| 参数名 | 说明 | 默认值 | 数据类型 |
| --- | --- | --- | --- |
| angle | 角度 | - | `Number` |
| isRound | 是否对结果进行四舍五入取整| - | `Boolean` |

## 函数返回
| 说明 | 数据类型 |
| --- | --- |
| 角度转弧度 | number |

## 实例 
``` javascript
let rad = angle2rad(90);
```
